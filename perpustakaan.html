Petunjuk untuk Admin Sekolah:
Saat ini link preview sudah menggunakan contoh buku yang dapat dilihat (demo). Untuk mengganti dengan koleksi asli milik sekolah:
1. Upload file PDF ke Google Drive.
2. Klik kanan file → "Bagikan" → ubah akses menjadi "Siapa saja yang memiliki tautan dapat melihat".
3. Salin ID file (contoh: 1r9A8YbeDtN96Bacv4O8IK9tZrNfbl0O-) lalu gunakan format:
https://drive.google.com/file/d/ID_FILE_ANDA/preview
4. Ganti atribut data-preview-url pada elemen .book-item sesuai buku masing-masing.

📌 Tata Tertib Perpustakaan Digital & Fisik: Dilarang membawa makanan/minuman, menjaga kebersihan, mengembalikan buku tepat waktu, merawat koleksi perpustakaan, serta menggunakan akun belajar untuk mengakses buku digital.

Gunakan fitur preview buku untuk membaca langsung dari Google Drive. Siswa dapat mengakses tanpa login (telah disetting publik).

Panduan & Akses Cepat

  1. Pilih buku yang diinginkan pada koleksi "Buku Digital Unggulan".
  2. Klik tombol "Preview Buku" untuk membuka jendela baca interaktif.
  3. Buku akan ditampilkan dalam tampilan PDF (zoom, scroll, dan dapat diunduh jika diizinkan).
  4. Untuk koleksi lengkap sekolah, kunjungi tautan Google Drive di bawah.
